# Day 18: Entry Functions ve Nesne Güncelleme
Bugün blockchain üzerindeki objeleri parametre olarak alıp durumlarını (state) güncellemeyi öğrendik.

### Öğrenilenler:
- **&mut Object**: Objeleri değiştirilebilir referans olarak alma.
- **State Management**: Zincir üzerindeki verinin kalıcı olarak güncellenmesi.
- **Entry Functions**: Dış dünyadan gelen parametrelerle akıllı kontrat etkileşimi.
