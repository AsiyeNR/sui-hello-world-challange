# Day 19: Sorgu Fonksiyonları ve Salt Okunur Erişim
Bugün Sui Object'leri üzerinden veri okumak için en verimli yol olan `&Farm` (immutable reference) kullanımını öğrendik.

### Öğrenilenler:
- **Immutable References**: Veriyi değiştirmeden güvenli sorgulama.
- **Getter Functions**: Struct içindeki özel alanları dışarıya açma.
- **Sorgu Performansı**: Neden `&mut` yerine `&` tercih edilmeli?
