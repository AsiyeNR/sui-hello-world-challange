# Day 13: Aggregation (Veri Özetleme)
Bugün `while` döngülerini kullanarak koleksiyonlar üzerinde istatistiksel hesaplamalar yapmayı öğrendik.

### Öğrenilenler:
- **total_reward**: Tüm ödüllerin toplamını hesaplama.
- **completed_count**: Belirli bir koşula (Enum durumu) uyan elemanları sayma.
- Vektör üzerinde iterasyon mantığı.
