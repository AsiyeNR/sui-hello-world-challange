# Day 06: String Tipi ve Metin Dönüşümleri
Bugün Move'da ham byte verilerini (`vector<u8>`) anlamlı metinlere (`String`) dönüştürmeyi öğrendik.

### Öğrenilenler:
- **std::string**: Standart kütüphane metin desteği.
- **string::utf8**: Byte dizisinden güvenli metin oluşturma.
- **Semantic Typing**: Veriyi daha anlamlı tiplerle sarmalamanın önemi.
