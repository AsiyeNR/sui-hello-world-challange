# Day 20: Event Emit (Olay Yayınlama)
Bugün blockchain dışındaki dünyayla haberleşmenin yolu olan Event mekanizmasını öğrendik.

### Öğrenilenler:
- **sui::event**: Olayları yöneten Sui modülü.
- **event::emit**: Bir olayı blockchain loglarına kaydetme.
- **Event Structs**: Mesajların `copy` ve `drop` yetenekleriyle tasarlanması.
